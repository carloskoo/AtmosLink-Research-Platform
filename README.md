# Weather Station Research

Sistema de adquisición de datos meteorológicos para el estudio de radioenlaces rurales en zonas altoandinas.

## Investigador Principal

Carlos Jesús Koo Labrín

Universidad Nacional de Cajamarca

## Objetivos

- Medición meteorológica local
- Comparación con ERA5
- Comparación con NASA POWER
- Correlación con RSSI
- Correlación con SNR
- Modelado predictivo mediante IA
